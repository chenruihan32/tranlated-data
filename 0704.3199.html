Abstract

In this paper, the stability condition for low-density parity-check (LDPC) codes on the binary erasure channel (BEC) is extended to generalized LDPC (GLDPC) codes and doubly-generalized LDPC (D-GLDPC) codes. It is proved that, in both cases, the stability condition only involves the component codes with minimum distance 2. The stability condition for GLDPC codes is always expressed as an upper bound to the decoding threshold. This is not possible for D-GLDPC codes, unless all the generalized variable nodes have minimum distance at least 3. Furthermore, a condition called derivative matching is defined in the paper. This condition is sufficient for a GLDPC or D-GLDPC code to achieve the stability condition with equality. If this condition is satisfied, the threshold of D-GLDPC codes (whose generalized variable nodes have all minimum distance at least 3) and GLDPC codes can be expressed in closed form.

I Introduction

A traditional LDPC code [1] of length N𝑁N and dimension K𝐾K is graphically represented through a bipartite graph with N𝑁N variable nodes and M≥N−K𝑀𝑁𝐾M\geq N-K check nodes [2]. A degree-n𝑛n check node of an LDPC code can be interpreted as a length-n𝑛n single parity-check (SPC) code, i.e. as a (n,n−1)𝑛𝑛1(n,n-1) linear block code, while a degree-n𝑛n variable node can be interpreted as a length-n𝑛n repetition code, i.e. as a (n,1)𝑛1(n,1) linear block code.

Doubly-generalized LDPC (D-GLDPC) codes, recently introduced in [3][4], extend the concept of LDPC codes, by allowing some variable and check nodes to be generic linear block codes instead of repetition and SPC codes respectively. If all the variable nodes are repetition codes, then the code is a generalized LDPC (GLDPC) code [2, 5, 6, 7, 8]. The codes used as variable and check nodes are called component codes of the D-GLDPC code; they will be supposed to have minimum distance dmin≥2subscript𝑑2d_{\min}\geq 2 [4]. The variable and check nodes which are not, respectively, repetition or SPC codes, are referred to as generalized nodes. The corresponding code structure is represented in Fig. 1. An (n,k)𝑛𝑘(n,k) generalized variable node is connected to n𝑛n check nodes; k𝑘k of the N𝑁N D-GLDPC encoded bits are received by the generalized variable node, and interpreted as its k𝑘k information bits. III Independent Sets and Minimum Distance

The development of a generalized stability condition for GLDPC and D-GLDPC codes is mostly based on a theorem proposed in this section. This theorem establishes a sufficient condition for a (k×(n−t))𝑘𝑛𝑡(k\times(n-t)) binary matrix, obtained by selecting n−t𝑛𝑡n-t columns in the generator matrix 𝐆𝐆\mathbf{G} (any representation) of a (n,k)𝑛𝑘(n,k) linear block code, to have rank equal to k𝑘k. The results presented in this section can be also developed from [12]. Here, they are independently proved and formulated for the scope of the paper.

Definition 1 (independent set)

Given a (k×n)𝑘𝑛(k\times n) binary matrix of rank r𝑟r, a set of t𝑡t columns is said an independent set when removing these t𝑡t columns from the matrix leads to a (k×(n−t))𝑘𝑛𝑡(k\times(n-t)) matrix with rank r−Δ​r<r𝑟Δ𝑟𝑟r-\Delta r<r, for some 0<Δ​r≤t0Δ𝑟𝑡0<\Delta r\leq t. The number t𝑡t is the size of the independent set.

If j𝑗j columns are an independent set for a certain representation of the generator matrix, they are an independent set for any other representation. Moreover, removing them from any representation of the generator matrix leads to the same rank reduction Δ​rΔ𝑟\Delta r. This is because all the possible representations of the generator matrix can be obtained by row summations starting from any other representation.

Lemma 1

If any representation of the generator matrix of a (n,k)𝑛𝑘(n,k) linear block code has an independent set of size j𝑗j, then the code minimum distance satisfies dmin≤jsubscript𝑑𝑗d_{\min}\leq j.

Proof:

Let us suppose that j𝑗j columns of a generator matrix 𝐆𝐆\mathbf{G} are an independent set of size j𝑗j. Then, it must be possible to perform row summations on 𝐆𝐆\mathbf{G} in order to obtain a new generator matrix representation 𝐆′superscript𝐆′\mathbf{G}^{\prime}, in which a certain number α𝛼\alpha of rows have all their 1’s in correspondence of only the columns of the independent set (see for example Fig. 2, where the first j𝑗j columns are supposed an independent set, and where 𝐀𝐀\mathbf{A}, 𝐁𝐁\mathbf{B} and 𝐂𝐂\mathbf{C} are non-null matrices). VI Conclusion

In this paper, a stability condition on the BEC has been derived for GLPDC and D-GLDPC codes, generalizing the inequality q∗≤[λ′​(0)​ρ′​(1)]−1superscript𝑞superscriptdelimited-[]superscript𝜆′0superscript𝜌′11q^{*}\leq[\lambda^{\prime}(0)\rho^{\prime}(1)]^{-1}, valid for standard LDPC codes. A derivative matching condition has been also defined, sufficient to achieve the stability condition with equality.

As for LDPC codes, only the variable and check nodes with minimum distance 2 are involved in the stability condition. The stability condition for GLDPC codes can be always explicitly expressed as an upper bound on the decoding threshold. D-GLDPC codes do not share this property in general; however, if all the generalized variable nodes have minimum distance at least 3, the stability condition becomes the same as for GLDPC codes. As a consequence, for GLDPC codes and for D-GLDPC codes with variable component codes of minimum distance at least 3 the decoding threshold assumes a simple closed form.
