1. Introduction

Temporal logics are well-established tools for the specification of correctness properties and their verification in hard- and software design processes. One of the most famous temporal logics is Kozenâ€™s modal Î¼ğœ‡\mu-calculus â„’Î¼subscriptâ„’ğœ‡\mathcal{L}_{\mu} [Kozen83] which extends multi-modal logic with extremal fixpoint quantifiers. â„’Î¼subscriptâ„’ğœ‡\mathcal{L}_{\mu} subsumes many other temporal logics like PDL [Fischer79] as well as CTLâˆ— [Emerson:1986:SNN], and with it CTL [EmersonHalpern85] and LTL [Pnueli:1977]. Hence, properties expressed by formulas of the modal Î¼ğœ‡\mu-calculus are only regular. There are, however, many interesting correctness properties of programs that are not regular. Examples include uniform inevitability [Emerson:1987:UIT] which states that a certain event occurs globally at the same time in all possible runs of the system; counting properties like â€œat any point in a run of a protocol there have never been more send- than receive-actionsâ€; formulas saying that an unbounded number of data does not lose its order during a transmission process; or properties making structural assertions about their models like being bisimilar to a linear time model.

When program verification was introduced to computer science, programs as well as their correctness properties were mainly specified in temporal logics. Hence, verification meant to check formulas of the form Ï†â†’Ïˆâ†’ğœ‘ğœ“\varphi\to\psi for validity, or equally formulas of the form Ï†âˆ§Ïˆğœ‘ğœ“\varphi\wedge\psi for satisfiability. An intrinsic problem for this approach and non-regular properties is undecidability. Note that the intersection problem for context-free languages is already undecidable [barhillel-etal:1961a].

One of the earliest attempts at verifying non-regular properties of programs was Non-Regular PDL [JCSS::HarelPS1983] which enriches ordinary PDL by context-free programs. Non-Regular PDL is highly undecidable, hence, the logic did not receive much attention for program verification purposes. Its model checking problem, however, remains decidable on finite transition systems â€“ it is even in P [lange-mcpdl].

Another example is Fixpoint Logic with Chop, FLC, [Mueller-Olm:1999:MFL] which extends â„’Î¼subscriptâ„’ğœ‡\mathcal{L}_{\mu} with a sequential composition operator. It is capable of expressing many non-regular â€“ and even non-context-free â€“ properties, and its model checking problem on finite transition systems is decidable in deterministic exponential time [ls-fossacs02]. It also properly subsumes Non-Regular PDL [langesomla-ipl06].

In order to achieve non-regular effects in FLC, the original â„’Î¼subscriptâ„’ğœ‡\mathcal{L}_{\mu} semantics is lifted to a function from sets of states to sets of states. This idea has been followed consequently in the introduction of Higher-Order Fixpoint Logic, HFLHFL\mathrm{HFL}, [viswanathan:hfl] which incorporates a simply typed Î»ğœ†\lambda-calculus into the modal Î¼ğœ‡\mu-calculus. Still, HFLHFL\mathrm{HFL}â€™s model checking problem on finite transition systems remains decidable. This has been stated in its introductory work [viswanathan:hfl]. It is also known that model checking HFLHFL\mathrm{HFL} is non-elementary with the following complexity bounds [langesomla-mfcs05].
